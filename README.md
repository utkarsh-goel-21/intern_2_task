# InAmigos Foundation Awareness Webpage

A responsive awareness webpage created for Task 1: NGO Awareness Webpage Creation. The page highlights InAmigos Foundation's purpose, ongoing projects, social impact, campaign hashtags, visuals, and volunteer call-to-action.

## Features

- Introduction to InAmigos Foundation
- Ongoing project details
- Social impact and purpose section
- Local images and visuals
- Volunteer/support call-to-action
- Responsive layout for desktop, tablet, and mobile
- Mobile-friendly header with a compact menu
- Browser tab logo/favicon
- Lightweight CSS animations

## How to View Locally

Open `index.html` directly in any browser.

No build tools, frameworks, or server setup are required.

## Vercel Deployment

Use these settings on Vercel:

- Framework Preset: `Other`
- Build Command: leave empty
- Output Directory: leave empty
- Root Directory: the folder containing `index.html`

After deployment, share the production URL ending in `.vercel.app`.

If visitors are asked to sign in, disable Vercel Authentication from:

`Project Settings` → `Deployment Protection` → `Vercel Authentication` → `Off`
